# **App Name**: MajorStockX

## Core Features:

- User Authentication & Management: Google Login and Email/Password registration/authentication. Admins can manage users, roles, and permissions (sales, production, inventory, reports).
- Centralized Database with Real-time Sync: Shared online database ensuring real-time data synchronization across all devices. Offline functionality with automatic synchronization upon reconnection and a manual 'Force Sync' button. Visual indicator for connection status (Online/Offline/Syncing).
- Inventory Management: Searchable inventory with filtering by category. Visual alerts for low or critical stock levels.
- Sales & Production Recording: Record sales with automatic stock deduction and printable transport guide generation. Record production with automatic stock increase.
- Printable Physical Count Form: Generates a printable form for physical stock counting, featuring a modern design and repeating headers.
- In-App Notifications: Alerts users about low stock, new sales, completed production, and other important movements using in-app notifications.

## Style Guidelines:

- Primary color: A deep, desaturated blue (#3498db) to evoke a sense of trust and stability, reflecting the construction materials industry.
- Background color: A light gray (#f0f2f5), creating a clean and professional backdrop.
- Accent color: A vibrant orange (#e67e22) is used for interactive elements and important alerts, adding a touch of energy.
- Body text: 'PT Sans', sans-serif.
- Headline font: 'Space Grotesk', sans-serif, which offers a modern and slightly technical feel.
- Mobile-first design, ensuring a responsive and intuitive interface with horizontal swipe navigation on mobile devices. Initial zoom is optimized for small screens.
- Clean, minimalist icons to represent product categories and actions, enhancing usability.
